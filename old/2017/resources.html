Computer Science Resources



Python
  • Artemis Python Cheat Sheet
  • Codecademy Python: a fun way to learn Python and get feedback and badges
  • CodingBat: a free site to practice Python coding skills
  • How to Think Like a Computer Scientist
  • Learn X in Y Minutes Python: a quick walk through of common Python expressions, good for reviewing the language
  • CheckiO: a fun adventure game to build Python skills
  • PyGame (advanced): a set of Python modules designed for writing games
Programming
  • Sublime Text: a text editor for coding, markup, and web design (not free, but has a free trial)
  • Gedit: a free and more lightweight text editor
  • Codepen: an online web design text-editor
  • HackThisSite: a free, safe, and legal site to test and expand hacking skills
  • Scratch and Snap: two visual, drag-and-drop programming languages for creating animations and games
  • Project Euler: a serious of challenging mathematical/computer programming problems
  • A Beginner's Guide to Programming Languages: a high-level overview of some of the more commonly-used languages
HTML/CSS
  • Artemis HTML/CSS Cheatsheet
  • Mozilla tutorials: HTML and CSS
  • w3schools tutorials: HTML and CSS
  • Hexadecimal Color Picker
  • Color scheme picker
  • Google Fonts
  • Dirty Markup: organizes and nicely formats HTML and CSS code
  • Tumblr Layouts: use HTML and CSS to build custom Tumblr themes

Enrichment Resources



Photoshop and Graphics
  • Gimp: a free open source graphics program similar to Photoshop
  • Inkscape: a free vector graphics program similar to Illustrator (Photoshop's cousin)
  • Blender: a free 3-d modeling program
  • Processing: a graphics programming language and community
  • Alice: an educational software that teaches students computer programming in a 3D environment
Summer Programs
  • Spira: a Brown-run engineering summer camp for rising 10th grader girls
  • GirlsGetMath@ICERM: a Brown-run summer math camp for rising 10th and 11th grade girls
  • Girls Who Code: a free, seven week immersive summer program on computer science, robotics, algorithms, web design, and mobile development for rising 11th and 12th graders
  • MIT's Women's Technology Program: a free MIT-run four-week summer academic and residential experience for female rising 12th graders to explore engineering
  • MITES offers free academic enrichment programs for rising 12th graders from across the country
  • CSSI: a free summer program at Google for rising freshman in college (apply in 12th grade)
Women in Technology:
  • The Anita Borg Institute for Women in Technology

  • Society of Women Engineers: a community for women computer scientists and engineers with opportunities for college scholarships
