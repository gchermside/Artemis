The Artemis Project 2000: Project Specifications

Specifications for Web Page

Your web page must have all of the following attributes. You can certainly have more than these, but these are the minimum requirements. You must also have at least three different pages that are in addition to your home page (different files). Your home page should have links to these pages and they should all have links back to it.
  • backgrounds for each page
  • titles for each page
  • an image
  • another image that is a link
  • two additional text links
  • an ordered list
  • an unordered list
  • a table
  • three different font sizes
