Print and Text
PRINT  This command is followed by one parameter.  It will be
             printed out below the command in the input window. The
             following example will display ’60’ in the window.
PRINT 60
Text  in Logo is fairly simple:  If you want Logo to recognize a word
         as just text and not as a command, it should follow a
         quotation mark.  The following command will display ‘Artemis’
         in the window.
PRINT “Artemis
For more than one word, just enclose the text in square brackets
PRINT [More than one word]
