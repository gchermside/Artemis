Variables
A VARIABLE is a symbol representing something.  This “something” in Logo can be a number, or some text.
Here is how to DEFINE a variable:
MAKE tells Logo you want to define a variable
foo is the name of the variable
The value of foo is now 90
MAKE “foo 90
When you define a variable, the name comes after a quotation mark.  When you use a variable, however, the name comes after a colon.
