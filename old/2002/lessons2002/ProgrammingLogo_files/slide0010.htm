More Conditional Statements
What if you wanted something to happen when the CONDITION is true AND something else to happen when it is false?
IFELSE  This command is followed by 3 parameters.  The first is the
               CONDITION, and the other two are CONSEQUENCES.
               The first consequence occurs if the condition is true.  The
               second consequence occurs if the condition is false.
IFELSE ( :foo > :bar ) [ PRINT [foo is greater] ]  [ PRINT [bar is greater] ]
