  • Once you have already saved a file, clicking Save will save it with the same name.
  • Save As allows you to save it as another name.  Save As will create another copy of your file.

File Extensions
  • Every file in Windows has a 3 letter extension at the end of it.
  • The extension tells you what program in which to open the file.
  • File extensions are like last names.  They identify the type of file.
13
Examples of Extensions
  • MP3 music files have the extension .mp3
  • Word files have the extension .doc
  • Notepad files use .txt
  • Images are .gif or .jpg (more on this later)
  • Paint files are .bmp
  • Executable files are .exe
  • Logo files are .lgo
  • C files are .c (you’ll use this for Robots)
  • Hair extensions

Mouse Buttons
  • Mice have two or three buttons.  Sometimes the middle button is a wheel.
  • The left button is used for selecting.
    • Opening programs requires double clicking the left button.
    • In word processing programs, dragging the mouse while holding the left button highlights text.
  • The right button shows options.
  • The wheel is used for scrolling.
