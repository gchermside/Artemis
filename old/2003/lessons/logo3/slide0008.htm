Many Parameters
What if you want your program to make a rectangle?  Then you would want to specify a width and a height.  This means you want to have two PARAMETERS.
Fortunately, this is easy!  You would simply add the second parameter right after the first. 
Try writing a program to make a rectangle that takes two parameters, a width and a height.  Test it out to see if it draws as you expected.
