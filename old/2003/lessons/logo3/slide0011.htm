Random
You can use RANDOM in many ways:
SETPENCOLOR RANDOM 20
SETSCREENCOLOR RANDOM 10
FORWARD RANDOM 300
RANDOM  This command is followed by one parameter (which is
                  a number).  It will give you a random number less
                  than the parameter.
This example will “return” a number less than 50:
RANDOM 50
 Try a few of these for yourself.
