Basic Logo Commands
FORWARD  This command is followed by a number.  The larger the number
                     the farther the turtle will go.  If you use a large enough number,
                     the turtle will fall off the screen and wrap around to the other side.
FORWARD 40
BACK  Follow this command with a number, as you did for FORWARD.  This
            time the turtle will move backwards.
BACK 33
RIGHT  This command is also followed by a number.  The number represents
             how many degrees you want the turtle to turn.  The example below will
             turn the turtle to the right 90 degrees.
RIGHT 90
LEFT  This command works exactly like the RIGHT command, only it turns the
           turtle to the left.
