Applications for Artemis 2006 are ready to be downloaded!

How to apply:

To apply for Artemis, you must complete the Student Application and submit one Teacher Recommendation. Both of these need to be postmarked by April 19th. 



How to contact us:

You can send us your application by mail or by e-mail.
Please mail applications to:

The Artemis Project 2006
Computer Science Department
Brown University
Box 1910
Providence, RI 02912-1910

Our e-mail address is artemis@cs.brown.edu. Please submit applications as .doc, .txt, or .rtf files.
