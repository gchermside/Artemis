The Artemis 2010 Curriculum!

Week 1: June 28-July 2

Computer-based:

  • Microsoft Excel: Grade Calculator
  • Microsoft PowerPoint: About Me PowerPoint
  • Adobe Illustrator: Name Tag, Poster
  • Adobe Photoshop: Advertisement
Non computer-based:
  • Graph Theory (Part 1): Euler Circuit & Path, Seven Bridges of Konigsberg, Handshaking Lemma
  • Linguistics
  • Graph Theory (Part 2): Prim's & Kruskal's MST Algorithms
  • Graph Theory (Part 3): Graph Coloring, Minimum Dominating Set

Week 2: July 6-July 9

Computer-based:

  • Web Design: HTML & CSS
Non computer-based:
  • Binary
  • Logic Gates

Week 3: July 12-July 16

Computer-based:

  • Introduction to Object-oriented Programming: Alice
  • Programming in Java: Basic Syntax, Inheritance, Tic-Tac-Toe, Fortune Teller
Non computer-based:
  • NIM Strategy
  • Search Algorithms: Breadth-First Search, Depth-First Search
  • Recursion: Towers of Hanoi

Week 4: July 19-July 23

Computer-based:

  • Robotics: Programming Finch Robots
Non computer-based:
  • Sort Algorithms: Selection, Quicksort, Insertion, Bubble, Mergesort
  • Circuits
  • Hardware

Week 5: July 26-July 30

Computer-based:

  • Final Projects in Java
Non computer-based:
  • Preparation for Final Presentations
