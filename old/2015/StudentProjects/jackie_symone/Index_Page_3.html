THE COLOR GAME

A color name is displayed on the screen. The text of the color is printed in a different color to the color name displayed. You have three seconds to identify the color the word was written in.

For Example:

'Yellow'

The user would select 'Green' in the time alotted to win. If the user chose another color such as 'Yellow' or 'Blue' the user would loose.
