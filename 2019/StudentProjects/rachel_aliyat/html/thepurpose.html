The Purpose

Although PoLITics is a student project, we believe that it has very real significance in today's world. Obligation was not the only reason why we chose to create this application and website. We really care about this issue, as it affects nearly every American's life today, as well as the lives of people around the world.

But what is this "issue", exactly?

Young people today are becoming increasingly detached from politics and what is going on around them, and we want to change that. Considering that our generation will run the country in the not-so-distant future, it is important that we become educated and understand some important political issues.

PoLITics is designed to encourage learning in a fun and safe environment. Everything is individual, so the user can utilize this tool however they wish to learn. In addtion it educates the user on which sources are trustworthy and which are not.
