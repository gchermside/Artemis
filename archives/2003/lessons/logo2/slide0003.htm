Conditional Statements
IF  This command is followed by 2 parameters.  The first is a
     CONDITION.  The other is a CONSEQUENCE. This command
     creates what is known as a CONDITIONAL STATEMENT. 
Condition
( > means greater than)
Consequence
The example above works like this:
If the value of foo is greater than the value of bar
then “foo is greater than bar” is printed
If the value of foo is not greater than the value of bar
nothing happens
IF ( :foo > :bar )  [ PRINT [foo is greater than bar] ]
