Parameters
When we used the basic Logo commands like FORWARD and RIGHT, we said that they were followed by a number.  This number is known as a PARAMETER.
The PARAMETER for SETPOS is two numbers in square brackets
The PARAMETERS for REPEAT are a number and then a command (or several commands) in square brackets
The PARAMETER is the number 50
Here are a few examples of PARAMETERS we have already seen:
FORWARD 50
REPEAT 4 [FORWARD 100 RIGHT 90]
SETPOS [100 100]
