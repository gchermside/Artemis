Fit or Fat

In this game, the user will be given a set of ten questions. Each question has to do with making the right choices concerning their health. Each answer the user picks will affect an avatar that represents them. With every wrong answer, the avatar becomes less healthy, gains weight, and develops medical issues. With every right answer the user picks, the avatar slims down, looks fitter, and appears happier.
