PoLITics

PoLITics is an application which can be downloaded on multiple devices. It is appropriate for all ages, especially teens and young adults - however, it is advised that younger children be supervised while playing. The app is designed to help young pepole understand how politics affects their daily lives. It includes one game involving trivia (and more serious issues). And a page was constructed to show the funnier side of politics and the people who run our country.

How does it work?

Trivia

The user is presented with questions of varying difficulty about political figures and issues. With each question succesfully answered, the user gains points. If the user cannot answer a question they will be linked to outside sources, encouraging them to learn more. We are careful to warn users of the credibility of the sources provided, and how to use them properly.

So Random

The user will be given a list of random and hilarious facts about politics and politicians.
